# OpenCode Prompt Recipes

## Bug Fix
```
Fix the bug in [file:function] where [symptom].
Error message: [paste error]
Expected behavior: [describe]
The root cause is likely in [area] based on [evidence].
```

## Feature Implementation
```
Implement [feature] in [file/module].
Requirements:
- [requirement 1]
- [requirement 2]
Follow the existing patterns in [reference file].
Add tests in [test file].
```

## Code Review Investigation
```
Investigate [file/module] for potential issues with:
- [concern 1]
- [concern 2]
Focus on [specific area] and report findings.
```

## Refactoring
```
Refactor [file/module] to [goal].
Current structure: [describe]
Desired structure: [describe]
Constraints: Do not change [protected areas].
Ensure all existing tests pass.
```

## Debugging
```
Debug why [test/feature] fails with [error].
Reproduction: [steps]
The failure started after [change/commit].
Check [files] for the root cause.
```
